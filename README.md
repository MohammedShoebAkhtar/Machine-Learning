# Random Forest Ensembles: The Effect of Number of Trees on Model Stability

## Overview
This tutorial demonstrates how the number of trees in a Random Forest model affects prediction accuracy and model stability. The study uses a heart disease dataset to illustrate the behaviour of ensemble models.

The project was created as part of a Machine Learning tutorial assignment.

## Dataset
The dataset contains medical information related to heart disease risk including age, cholesterol level, blood pressure, and lifestyle factors.

Target variable:
Heart Disease Status

## Tutorial Content
The tutorial covers:

- Data preprocessing
- Random Forest model training
- Experimentation with different numbers of trees
- Model stability analysis
- Feature importance analysis
- Model evaluation using confusion matrix and classification metrics
